# Follow-Up Task

Check on outstanding outreach and send follow-ups.

## Steps

1. Recall all prospects from memory with tag "prospect:contacted"
2. Check email for any replies from prospects
3. For replies:
   - Update prospect status to "prospect:replied"
   - Analyze sentiment (interested, not interested, needs info)
   - Draft appropriate response
4. For non-replies (contacted 3+ days ago, fewer than 3 follow-ups):
   - Draft a follow-up email with a different angle/value prop
   - Send the follow-up
   - Update memory with follow-up count
5. Summarize: prospects contacted, replies received, follow-ups sent

# Prospect Discovery Task

Search for new prospects matching our ideal customer profile.

## Steps

1. Use the browser to visit target directories and websites
2. For each prospect found, extract:
   - Company name
   - Contact name and title
   - Email address
   - Website URL
   - Brief description of their business
3. Save each prospect to memory with tag "prospect:new"
4. Skip any prospect already in memory
5. Aim to find 10-20 new prospects per run

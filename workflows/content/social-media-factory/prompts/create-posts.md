# Weekly Social Media Content Batch

Create a week's worth of social media content.

## Steps

1. Recall brand voice and recent posts from memory
2. Research trending topics in the niche using browser
3. Plan the week's content mix:
   - 5 Twitter/X posts
   - 3 LinkedIn posts
   - 3 Instagram captions
4. For each post:
   - Write platform-optimized copy
   - Suggest a visual concept
   - Include relevant hashtags
   - Note the best posting day/time
5. Compile all posts into a structured email
6. Send via email with subject: "[Social] Week of [DATE] — Content Batch"
7. Save the batch to memory with tag "social:batch"

## Email Format

Organize posts by day, then by platform:

**Monday:**
- Twitter: [post text] | Suggested time: 9am
- LinkedIn: [post text] | Suggested time: 8am

**Tuesday:**
...

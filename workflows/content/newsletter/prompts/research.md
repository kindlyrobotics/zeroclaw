# Newsletter Research Task

Gather material for this week's newsletter edition.

## Steps

1. Browse configured news sources for trending stories in our niche
2. Use HTTP requests to check RSS feeds if available
3. For each relevant story, extract:
   - Headline and source URL
   - Key facts and data points
   - Why it matters to our audience
4. Recall from memory what we covered in recent editions — avoid repeats
5. Save all research findings to memory with tag "newsletter:research"
6. Select the top 5 stories and 5 quick hits
7. Save the curated list to memory with tag "newsletter:curated"

## Selection Criteria

- Relevance to our audience
- Timeliness (prefer this week's news)
- Actionability (can readers do something with this?)
- Uniqueness (not widely covered elsewhere)

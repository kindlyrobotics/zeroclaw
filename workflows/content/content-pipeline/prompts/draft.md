# Content Drafting Task

Check for pending content briefs and produce drafts.

## Steps

1. Recall pending content briefs from memory (tag "content:brief")
2. For each brief:
   a. Research the topic using browser and HTTP tools
   b. Create an outline with 4-6 main sections
   c. Write the full draft (target length from brief, default 1500 words)
   d. Save the draft to a file in the workspace
3. Email the finished drafts to the client
4. Update memory: move brief from "content:brief" to "content:delivered"
5. Log delivery date and word count

## If No Pending Briefs

- Report "No pending content briefs" and exit
- Check memory for any overdue deliverables

# Quiz Generation

Create and administer quizzes for knowledge assessment.

## Quiz Types

### Quick Check (3-5 questions)
- After each lesson
- Multiple choice and short answer
- Tests immediate understanding

### Review Quiz (10 questions)
- Weekly or bi-weekly
- Covers material from several sessions
- Mix of difficulty levels

### Mastery Test (15-20 questions)
- End of topic/unit
- Must score 80%+ to advance
- Includes application problems

## Question Design

- Clear, unambiguous wording
- One concept per question
- Include distractors that test common misconceptions
- Progressive difficulty within each quiz

## Scoring and Feedback

- Score immediately after each answer
- Explain correct answer for wrong responses
- Save scores to memory (tag "quiz:[student]:[topic]")
- Adjust future lesson difficulty based on results
- Celebrate high scores, encourage on low scores

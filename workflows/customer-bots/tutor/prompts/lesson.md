# Lesson Delivery

Guide for structuring a tutoring session.

## Before Starting

1. Recall student profile from memory (tag "student:[name]")
2. Check what was covered last session
3. Identify the next topic in their learning path

## During Lesson

1. **Warm-up** (2 min) — Quick review question from last session
2. **Teach** (5 min) — Explain the new concept clearly
3. **Practice** (5 min) — Work through 2-3 examples together
4. **Check** (3 min) — Student solves one independently
5. **Wrap** (2 min) — Summarize key points, assign practice

## If Student Struggles

- Step back to prerequisite concept
- Try a different explanation approach
- Use an analogy from their interests
- Break the problem into smaller steps

## After Session

Save to memory:
- Topics covered
- Student performance (understood / partial / struggled)
- Next session plan
- Tag: "lesson:[student]:[date]"

# Support Interaction Guide

Reference for handling common support scenarios.

## Common Categories

### Account Issues
- Password reset → Direct to reset flow
- Login problems → Check recent changes, browser/app version
- Account access → Verify identity, then assist or escalate

### Product Questions
- How to use X → Recall from memory, provide step-by-step
- Feature requests → Log in memory (tag "feedback:feature"), thank them
- Bug reports → Collect details (steps to reproduce, screenshots), log and escalate

### Billing
- Always escalate billing issues to the escalation agent
- Collect: account ID, charge in question, expected vs actual

### Technical Issues
- Collect: device, OS, app version, steps to reproduce
- Check memory for known issues (tag "kb:known-issue")
- If known, provide workaround
- If unknown, escalate with full details

## Resolution Logging

After resolving any issue, save to memory:
- Category and subcategory
- Problem description
- Solution provided
- Tag: "kb:solution"

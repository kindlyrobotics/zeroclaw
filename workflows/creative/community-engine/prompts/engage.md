# Community Engagement Task

Run daily engagement activities and monitor health.

## Steps

1. Check community activity since last run
2. Welcome any new members not yet greeted
3. Answer unanswered questions using knowledge base
4. Delegate today's engagement activity to `engagement` agent:
   - Monday: Discussion prompt
   - Tuesday: Quick poll
   - Wednesday: Tip/knowledge share
   - Thursday: Challenge/contest
   - Friday: Community highlight (top contributors)
   - Weekend: Casual/off-topic thread
5. Identify inactive members (no activity in 7+ days)
   - Send re-engagement message to top 3
6. Check for spam/scam messages — delegate to `moderator`
7. Log community metrics to memory (tag "community:metrics:[date]")

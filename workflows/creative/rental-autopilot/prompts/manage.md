# Daily Property Management Task

Check pricing, handle guest needs, and maintain property status.

## Steps

1. Pricing check:
   a. Browse competitor listings using browser
   b. Compare current pricing against competitors
   c. Check for local events or seasonal factors
   d. Generate pricing recommendation
   e. Save to memory (tag "pricing:[property]:[date]")
2. Guest communication check:
   a. Check for new inquiries or messages
   b. Respond to any unanswered messages
   c. Send check-in instructions to guests arriving tomorrow
   d. Send thank-you messages to guests who checked out today
3. Review management:
   a. Check for new reviews
   b. Draft responses (thank positive, address negative)
   c. Send drafts for approval via email
4. Maintenance check:
   a. Review any reported issues in memory
   b. Send status updates to owner if pending issues
5. Daily summary to owner via email:
   - Today's check-ins/check-outs
   - Pricing recommendation
   - New reviews
   - Any issues requiring attention

## Monitoring Targets

1. **Marketplaces** — Amazon, eBay, Etsy, Alibaba for counterfeit listings
2. **Social media** — impersonator accounts using brand name/logo
3. **Domains** — new registrations matching brand or typosquats
4. **App stores** — knockoff apps using brand name
5. **Google search** — ads and results using the trademark

## Infringement Severity

- **Critical**: counterfeit products for sale, active impersonation
- **High**: unauthorized trademark use in commercial context
- **Medium**: typosquat domains, misleading affiliates
- **Low**: fan pages, reviews using the mark (usually fair use)

## Rules

- Save all detected infringements to memory (tag "infringement:[id]")
- Track takedown status: detected → reported → pending → resolved
- Draft appropriate response (C&D, DMCA, platform report) per severity
- Never send legal threats without human approval — draft only
- Include screenshot evidence with every report

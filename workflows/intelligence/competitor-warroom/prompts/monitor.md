# Daily Competitor Monitoring Task

Check all tracked competitors for changes.

## Steps

1. Recall competitor list from memory (tag "competitor:tracking")
2. For each competitor:
   a. Visit their website — compare key pages against saved snapshots
   b. Check their careers/jobs page — note new or removed positions
   c. Browse their social media profiles — recent posts and engagement
   d. Search for recent news mentions
   e. Check review sites for new reviews
3. Compare all findings against previous day's snapshot in memory
4. For each change detected:
   - Classify: pricing / product / hiring / messaging / tech / sentiment
   - Assess strategic implication
   - Save updated snapshot to memory (tag "competitor:[name]:[date]")
5. If significant changes found:
   - Send alert via email with analysis
6. If no changes: log silently to memory

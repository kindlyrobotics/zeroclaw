# Morning Email Triage

Start-of-day email review and prioritization.

## Steps

1. Check email inbox for all unread messages
2. For each email, categorize:
   - **Urgent**: needs response today (deadlines, escalations, VIPs)
   - **Important**: needs response this week (projects, requests)
   - **FYI**: informational, no action needed
   - **Spam/Noise**: unsubscribe candidates
3. For urgent emails, draft a response
4. Compile daily briefing:

## Briefing Format

Send via Telegram:

```
Morning Briefing — [Date]

URGENT (respond today):
- [Sender]: [Subject] — [1-line summary]
  Draft: [your draft response]

IMPORTANT (this week):
- [Sender]: [Subject] — [1-line summary]

FYI:
- [count] informational emails (details in email)

TODAY'S TASKS:
- [recall active todos from memory]

REMINDERS:
- [upcoming deadlines from memory]
```

5. Save briefing to memory (tag "briefing:[date]")

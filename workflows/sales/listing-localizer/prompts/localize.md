# Listing Localization Task

Localize product listings for target markets.

## Steps

1. Receive product listing data (title, description, bullets, specs)
2. Identify target market(s)
3. For each target market:
   a. Research competitor listings using browser (top 5 similar products)
   b. Extract local keywords and phrasing patterns
   c. Translate and adapt all listing elements
   d. Convert units, sizing, and regulatory requirements
   e. Optimize title and keywords for local search
   f. Delegate to market-specific agent if available
4. Compile localized listings
5. Send via email with original and localized versions side-by-side
6. Save localization to memory (tag "listing:[sku]:[market]")

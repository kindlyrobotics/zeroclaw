# Influencer Discovery Task

Find new influencers matching campaign criteria.

## Steps

1. Recall campaign criteria from memory (tag "campaign:active")
2. Browse target platforms for influencers in the niche:
   a. Search relevant hashtags and topics
   b. Check "suggested" profiles from known good influencers
   c. Browse niche-specific directories
3. For each potential influencer:
   a. Extract: username, follower count, engagement rate, bio, recent posts
   b. Score using evaluation criteria (1-10)
   c. Check memory for duplicates (tag "influencer:discovered")
   d. Save to memory with score and profile data
4. Compile top 10 new discoveries
5. For each scoring 7+:
   - Draft personalized outreach email referencing a specific post
6. Send discovery report via email with profiles and draft outreach
7. Target: 15-25 new profiles evaluated per run

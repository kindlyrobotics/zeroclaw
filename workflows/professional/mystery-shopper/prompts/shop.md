# Mystery Shopping Audit Task

Perform UX audit on configured competitor sites.

## Steps

1. Recall competitor list from memory (tag "competitor:sites")
2. For each competitor site:
   a. Visit homepage — note load time, first impression
   b. Test search — search for 3 relevant terms
   c. Browse categories — navigate 2-3 category paths
   d. View product pages — check 3 products for completeness
   e. Test add-to-cart flow (do NOT purchase)
   f. Test checkout flow up to payment page (do NOT submit)
   g. Check for chat widget — note response time
   h. Test mobile viewport if possible
   i. Check for broken links and errors
   j. Take screenshots at each step
3. Score each area (1-10) per the audit checklist
4. Compare against previous audit in memory
5. Compile UX audit report:
   - Overall grade per competitor
   - Detailed scores by category
   - Key findings (positive and negative)
   - Trend comparison vs. last audit
   - Recommendations for our own site based on competitor best practices
6. Send report via email
7. Save audit to memory (tag "ux-audit:[competitor]:[date]")

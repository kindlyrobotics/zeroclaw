# Report Generation Task

Collect data and generate a structured report.

## Steps

1. Recall report configuration from memory (tag "report:config")
   - Data sources and metrics to collect
   - Report format and sections
   - Delivery recipients
2. Collect data:
   a. Browse configured web sources
   b. Query APIs via HTTP requests
   c. Read local data files if available
3. Process data:
   a. Normalize formats and units
   b. Calculate period-over-period changes
   c. Identify trends and anomalies
   d. Compare against previous report from memory
4. Generate report following template structure
5. Quality check:
   - All sections populated
   - Data points are consistent
   - Recommendations are actionable
6. Send via email
7. Save report data to memory (tag "report:[type]:[date]")

## Report Email Format

Subject: [Report Type] — [Period] — [Date]
Body: Full structured report

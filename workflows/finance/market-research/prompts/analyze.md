# Market Analysis Task

Conduct market research and deliver a report.

## Steps

1. Identify the target market/topic (from request or memory)
2. Research using browser:
   a. Market size and growth data
   b. Key players and market share
   c. Recent news and developments
   d. Industry trends and forecasts
3. Research using HTTP (if APIs available):
   a. Financial data
   b. Industry databases
4. Compile findings into structured report
5. Generate executive summary
6. Send report via email
7. Save research to memory (tag "research:[market]")

## Report Email Format

Subject: [Market Research] [Topic] — [Date]

Body: Full structured report following the template in identity.

Attachments: None (full report in email body)

## Quality Checklist

- [ ] Minimum 5 sources cited
- [ ] Quantitative data included
- [ ] Competitive landscape covered
- [ ] Actionable recommendations provided
- [ ] Executive summary is standalone-readable
